# **3.4 Sensor vs Transducer**

คำว่า **Sensor** และ **Transducer** มักถูกใช้แทนกันในชีวิตประจำวัน แต่ในทางวิศวกรรม ทั้งสองคำมีความหมายเฉพาะตัวและมีขอบเขตที่แตกต่างกันเล็กน้อย

## **3.4.1 Sensor (เซ็นเซอร์)**

อุปกรณ์ที่ทำหน้าที่ **รับรู้ (sense)** ปริมาณทางกายภาพ และ **แปลงเป็นสัญญาณที่สามารถนำไปใช้วัดหรือประมวลผลได้** โดยทั่วไปสัญญาณที่ออกมามักเป็นสัญญาณไฟฟ้า เช่น แรงดัน, กระแส, ความต้านทาน

### **คุณสมบัติของ Sensor**

- ทำหน้าที่ “ตรวจจับ” ปริมาณทางกายภาพ
- ให้สัญญาณออกที่สัมพันธ์กับสิ่งที่วัด
- มักเป็นอุปกรณ์ **Input** ของระบบ
- มักมีความไว (sensitivity) และช่วงวัด (range) ที่กำหนดชัดเจน

### **ตัวอย่าง Sensor**

- **Thermistor** → อุณหภูมิ → ความต้านทาน
- **LDR** → แสง → ความต้านทาน
- **Hall Sensor** → สนามแม่เหล็ก → แรงดัน
- **Accelerometer** → ความเร่ง → สัญญาณดิจิทัล/อนาล็อก

## **Transducer (ทรานสดิวเซอร์)**

อุปกรณ์ที่ทำหน้าที่ **แปลงพลังงานจากโดเมนหนึ่งไปยังอีกโดเมนหนึ่ง** ไม่จำกัดว่าต้องเป็น Input หรือ Output ไม่จำกัดว่าต้องเป็นไฟฟ้า → อาจเป็นเสียง, แสง, การเคลื่อนที่, ความร้อน ฯลฯ

### **คุณสมบัติของ Transducer**

- ทำหน้าที่ “แปลงพลังงาน” ระหว่างโดเมน
- อาจเป็น Input หรือ Output
- ไม่จำเป็นต้องให้สัญญาณไฟฟ้าเสมอไป
- เป็นคำที่มีความหมายกว้างกว่า Sensor

### **ตัวอย่าง Transducer**

#### **Input Transducer**

- **Piezoelectric** → แรงกด → ไฟฟ้า
- **Microphone** → เสียง → ไฟฟ้า
- **Photodiode** → แสง → กระแสไฟฟ้า

#### **Output Transducer**

- **Speaker** → ไฟฟ้า → เสียง
- **LED** → ไฟฟ้า → แสง
- **Motor** → ไฟฟ้า → การหมุน

# **ความสัมพันธ์ระหว่าง Sensor และ Transducer**

คุณสามารถสรุปความสัมพันธ์ได้แบบนี้

### **Sensor ⊂ Transducer**

เซ็นเซอร์คือ “ทรานสดิวเซอร์ชนิดหนึ่ง” ที่ถูกออกแบบมาเพื่อ **วัด** ปริมาณทางกายภาพ

แต่ทรานสดิวเซอร์มีความหมายกว้างกว่า เพราะรวมถึงอุปกรณ์ที่แปลงพลังงานเพื่อ **สร้างผลลัพธ์** เช่น ลำโพง, มอเตอร์, LED

# **สรุป**

| ประเภท                | หน้าที่                     | ตัวอย่าง                          |
| --------------------- | --------------------------- | --------------------------------- |
| **Sensor**            | รับรู้และวัดปริมาณทางกายภาพ | Thermistor<br>LDR<br>Hall Sensor  |
| **Input Transducer**  | แปลงพลังงาน → สัญญาณไฟฟ้า   | Piezo<br>Microphone<br>Photodiode |
| **Output Transducer** | แปลงไฟฟ้า → พลังงานรูปอื่น  | Speaker<br>LED<br>Motor           |
